# Python Calculator in Docker

Простое консольное приложение калькулятора, упакованное в Docker-контейнер.

## Функциональность

- Сложение
- Вычитание
- Умножение
- Деление

## Требования

- Docker
- Python 3.8+ (только для локальной разработки)
